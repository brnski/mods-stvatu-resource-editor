## Contributing

This is a side project maintained on a best-effort basis. Bug reports and pull requests are welcome.

If you'd like to contribute:

1. Fork the repo and create a branch from `main`.
2. Make your changes and test them in-game.
3. Open a pull request with a clear description of what changed and why.

There's no formal review SLA, but PRs will be looked at when time allows.
